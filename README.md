# Omnicontrol
一款硬件设备中控软件，主要用于控制公司自主研发的混合矩阵。项目采用MVP设计模式，分为八大功能模块。播放器使用ijkplayer框架（修改了ffmpeg源码进行优化），异步任务使用RxJava2.0实现。同时使用了ButterKnife、dom4j等第三方库。
